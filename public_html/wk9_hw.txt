2. Is it possible that someone querying the database at the same time could see your insert partially complete (eg. only see some fields filled in)? What database property is this related to?

No. This is related to the database property, ACID. A person querying the database is an isolated action from you inserting a new row into the database (Isolation). Your row insert will need to be 100% complete and valid before the other user can see what data you've added (Atomicity).
